# Ariadne

**Ariadne** is a Grasshopper plugin for the inverse design of form-found structures in Rhino 8. It uses the [force density method](docs/articles/theory.html) (FDM) via the Theseus solver for directed form-finding of tensile network structures.

- [**Theory**](docs/articles/theory.md) — Mathematical introduction to the force density method and optimization over force densities (Schek 1974).
- [**Getting started**](docs/articles/getting-started.md) — Install, first network, forward solve and optimization.
- [**Workflow**](docs/articles/workflow.md) — Visual workflow and main components.
- [**Component guide**](docs/articles/component-guide.md) — Reference for Ariadne components (Design, objectives, utilities).
- [**Examples**](docs/articles/examples.md) — Example Grasshopper files to download and try.
- [**API reference**](api/index.md) — C# API generated from the source.
